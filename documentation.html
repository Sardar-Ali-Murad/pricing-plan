Get YouTube Tags

Streamline tag retrieval for YouTube videos by providing a query.

Request Method

GET

Endpoint

https://tags-lime.vercel.app/youtubeTags/{query}

Example Response

Get TikTok Tags

Retrieve relevant tags for TikTok content by providing a query. .

Request Method

GET

Endpoint

https://tags-lime.vercel.app/tiktokTags/{query}

Example Response

            {
                "data": {
                    "query": "baby playing | baby dance | baby playing with toys |   baby in the pool",
                    "tags": [
                        "#baby",
                        "#babyannabell",
                        "#babydolls",
                        "#babydoll",
                        "#babyborn",
                        "#babysongs",
                        "#kidsplaying"
                    ]
                }
            }
